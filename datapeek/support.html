Frequently Asked Questions

How do I open a database file?

  1. Open DataPeek
  2. Click Open File or drag and drop a file
  3. Supported formats: SQLite, JSON, CSV, Parquet, Excel

How do I run SQL queries?

  1. Open a database file
  2. Click the Query tab
  3. Write your SQL query
  4. Press Cmd + Enter to execute

How do I upgrade to Pro?

  1. Open DataPeek
  2. Go to Settings → Upgrade
  3. Click Upgrade to Pro
  4. Complete purchase through App Store

How do I restore my purchase on a new device?

  1. Open DataPeek
  2. Go to Settings → Upgrade
  3. Click Restore Purchases
  4. Sign in with the same Apple ID used for the original purchase

How do I request a refund?

Refunds are processed by Apple:

  1. Visit reportaproblem.apple.com
  2. Sign in with your Apple ID
  3. Find your DataPeek purchase
  4. Select "Request a refund"

My file won't open

Try these steps:

  1. Verify the file format is supported (SQLite, JSON, CSV, Parquet, Excel)
  2. Check if the file is corrupted by opening it in another application
  3. Ensure the file is not locked by another process
  4. Try making a copy of the file and opening the copy

How do I export data?

  1. Run a query or select a table
  2. Click Export button
  3. Choose format (CSV, JSON, Excel)
  4. Select destination and save

How do I delete my data?

DataPeek stores all data locally on your device. To delete:

  1. Uninstall the app

DataPeek never uploads your data. Your files remain on your device.
